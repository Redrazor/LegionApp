## Post Body

Hi all,

I've put together a free web companion app for Star Wars: Legion and wanted to share it with the community here. It's the sister app to my Shatterpoint tool, ShatterApp.

**LegionApp** — [legionapp.com](https://legionapp.com) *(update once live)*

Runs in the browser, no account or install required. Installable as a PWA if you'd like a home-screen shortcut.

**What's in it:**

*Browse* — a faction-grouped, searchable, filterable grid of every unit. Tapping a card opens the full profile: card scan, a native stat block (defense / wounds / courage / speed / surge), the upgrade-slot bar, keywords with tap-to-define glossary popovers, and errata & points history.

*Build* — an army builder for Standard (800) and Skirmish (500). Slot-aware, faction-legal upgrade picker with live validation: rank limits, points ceiling, single-faction rule (mercenary allies handled), and duplicate-unique conflicts, each with a plain-English ✓/✕. Save multiple lists, share via URL, print a clean army sheet.

*Collection* — expansion ownership tracking by quantity with per-faction progress, an owned-only Browse filter, and JSON import/export backup.

*Reference* — keyword glossary, upgrade-slot / rank / faction reference, and an embedded rulebook viewer.

An in-game Play tracker with real-time multiplayer is in progress.

Happy to add things — let me know what would be genuinely useful.
