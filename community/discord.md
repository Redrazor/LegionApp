# Discord — The Legion Discord

## Server
**The Legion Discord** — https://discord.com/invite/swlegion (~11.7k members)
The main hub for Star Wars: Legion. Unofficial, community-run.

## Approach
1. Join and **read the rules** — find the self-promo / `#fan-projects` / `#tools` channel.
   Many servers restrict tool posts to a specific channel or weekly thread.
2. Be present for a bit before dropping a link. A few helpful replies first = warmer reception.
3. Post in the right channel, engage every reply for the first 24h.
